# Testing the Pale Blue Nexus static site

## Setup
- Static site, no build step. Cloudflare Pages serves the repo root (`wrangler.toml` → `pages_build_output_dir = "."`). No `package.json`.
- Serve locally from the repo root: `python3 -m http.server 8099` (run detached: `nohup python3 -m http.server 8099 --directory ~/repos/palebluenexus >/tmp/pbn-srv.log 2>&1 &`). Then open `http://localhost:8099/book/`.
- Maximize the browser before recording: `wmctrl -r :ACTIVE: -b add,maximized_vert,maximized_horz` (install wmctrl if missing).
- Quick sanity check without the browser: `curl -s -o /dev/null -w '%{http_code}' http://localhost:8099/book/` and hit a TikTok thumb like `http://localhost:8099/images/feed/tt-<id>.jpg` (both should be 200).

## What to verify on /book/
1. Homepage "Be a Guest" nav link routes to `/book/` (link is site-wide: index.html, partner/, episodes/, blog/).
2. Hero "Come on Pale Blue Nexus." + proof band (5K subs, 20K+ views, 4 cities, 4 channels).
3. Four Top Performing clip cards, auto-generated by `scripts/update-feed.mjs` (`bookTopHtml`) between the `<!-- AUTO-BOOK-TOP:start/end -->` markers. All thumbnails must load — especially the TikTok cards.
4. Section headers left-aligned and consistent (uses shared `.section-head`).
5. Huckabuy title-sponsor strip: eyebrow above logo; the strip links to `/partner/`.
6. Inline Calendly scheduler loads the real `calendly.com/yohann8/pbn` "Pale Blue Nexus Pre-Interview" event with a live date picker.
7. Footer soft CTA "See partnership tiers" links to `/partner/`.

## Gotchas / things that might break
- **TikTok thumbnail paths**: feed thumbs are stored relative (`images/feed/tt-*.jpg`). On a subdirectory page like `/book/` a relative path resolves to `/book/images/...` and 404s. `cardHtml()` in `scripts/update-feed.mjs` normalizes relative thumbs to root-absolute (`/images/...`) while preserving `http(s)://` and already-absolute paths. If TikTok thumbs show broken images on `/book/`, check this normalization and re-run the generator. YouTube thumbs are absolute `i.ytimg.com` URLs and are unaffected.
- **Regenerating locally without the API key**: `update-feed.mjs` reuses the committed `data/feed.json` when `ENSEMBLE_API_KEY` is absent and only rewrites `data/feed.json` when it actually fetches fresh data — so local HTML regen won't churn the feed. Run `node scripts/update-feed.mjs` to regenerate `index.html` and `book/index.html`.
- **Calendly scroll trap**: when scrolling past the embedded Calendly iframe, mouse-wheel scroll is captured by the iframe. To scroll the page itself, put the cursor over the dark page margin (e.g. far-left x≈90) and scroll there.
- **Calendly cookie banner** appears over the widget on first load; dismiss it ("I understand") to reveal the date/time picker for a clean screenshot.

## Devin Secrets Needed
